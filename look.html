การแสดง Animation
  1. Animation การปล่อยหมุด 
    map.Overlays.drop(MARKER)
    • MARKER: marker ที่ต้องการปล่อย
  2. Animation หมุดกระเด้ง 
    map.Overlays.bounce(MARKER)
  3. Animation การเลื่อนหมุด 
    maker.move(LOCATION, ANIMATE)
    • LOCATION: ตำแหน่งที่จะเลื่อนไป
    • ANIMATE: ให้มี animation หรือไม่
  4. Animation การเลื่อนหมุดตามเส้น 
    map.Overlays.pathAnimation(MARKER, PATH);
    • MARKER: หมุด
    • PATH: Polyline ของเส้นทางที่จะเลื่อน
